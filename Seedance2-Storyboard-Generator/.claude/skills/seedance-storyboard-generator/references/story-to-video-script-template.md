# Story-to-Video Script — Universal Conversion Tool

## Usage Guide

This tool helps you quickly convert any story (novel, short fiction, real events) into a professional shot-by-shot script that meets Seedance 2.0 AI video production standards.

**Important: Scripts produced by this tool must strictly follow the quality bar in [good-script-examples.md](good-script-examples.md).**

---

## Step 1: Core Hook & Story Analysis

### 1.1 Core Hook Positioning

Before converting, define the story's core hook (2–4 words):

| Hook Type | Example | Description |
|-----------|------|------|
| Desperate counterattack | Lin Chong at Wind and Snow Temple | Protagonist erupts after being cornered |
| Revenge satisfaction | The Count of Monte Cristo | Betrayal followed by revenge arc |
| Healing / warm | Doraemon | Heartwarming everyday story |
| Suspense / thriller | Steins;Gate | Plot full of mystery |
| Growth / inspirational | Slam Dunk | Protagonist grows through effort |
| Love story | Titanic | Romance as main axis |

### 1.2 Six Elements of Story Synopsis

Fill this table to build the full story framework:

| Element | Content | Example (Lin Chong) |
|------|----------|-------------|
| **Setting** | Time, place, era | Late Northern Song, Cangzhou prison camp, brutal winter blizzard |
| **Opening conflict** | What breaks status quo | Framed and exiled; guards hay depot unaware killers approach |
| **Protagonist portrait** | Who they are, core traits | Master fighter who endures all, wants only to finish sentence, near breaking point |
| **Main plot** | Core progression | Depot collapse → temple refuge → overhear plot → counterattack |
| **Ending** | Final outcome & change | Slays enemies, abandons return hope, joins outlaws on snowy night |

### 1.3 One-Line Selling Point

**Formula:** [Emotional call] + [core conflict] + [visual climax]

**Examples:**
- Lin Chong: Enough is enough—no more patience! On a night of wind and snow, Leopard Head Lin Chong draws his spear, blood splashes across the mountain temple!
- Sima Guang: Calm under pressure, clever rescue! Young Sima Guang smashes the jar to save a child, showing extraordinary composure!

## Step 2: Character Profile Construction

Every major character needs a full profile:

### Character Profile Template

```
Character name
Visual look: [age] + [appearance] + [clothing details] + [signature action/prop]
Background: [social role] + [relation to protagonist] + [past]
Core tags: [tag1] + [tag2] + [3–4 tags]
Personality: [normal state] → [under pressure] → [breakthrough/turn]
Signature line: [line that defines the character]
```

### Comparison

**❌ Simple version (not usable):**
```
Lin Chong: mid-30s, Imperial Guard commander, patient personality
```

**✅ Full version (standard format):**
```
Lin Chong
Visual look: Mid-thirties, weathered resolute face, worn cotton coat, felt hat, long spear, wine gourd at waist.
Background: Former Imperial Guard commander, now Cangzhou prisoner.
Core tags: Suppressed rage, tragic hero, killing god unleashed.
Personality: Once submissive and law-abiding; when cornered, erupts with explosive resolve.
Signature line: "This world forces good men to become demons!"
```

## Step 3: Script Outline (Setup → Rise → Turn → Resolution)

### Three-Act / Four-Act Structure

Choose structure by runtime:

| Runtime | Structure | Per-act content |
|------|------|----------|
| 5 eps × 15s | Simplified three-act | Setup (1) → Rise/Turn (2–3) → Climax/End (4–5) |
| 10 eps × 15s | Standard three-act | Setup (1–2) → Rise (3–6) → Turn/End (7–10) |
| 20 eps × 15s | Four-act | Setup (1–5) → Rise (6–10) → Turn (11–15) → End (16–20) |

### Outline Template

```
Act I (Setup): [scene establishment] + [protagonist status] + [inciting event]
Act II (Rise/Turn): [conflict develops] + [2–3 setbacks] + [turning point]
Act III (Climax): [final confrontation] + [emotional/action peak]
Act IV (Resolution): [ending state] + [theme lift] + [lingering resonance]
```

### Example (Lin Chong — 5-episode version)

```
Act I (Setup): Lin Chong guards hay depot alone; harsh environment, lonely oppressive mood.
Act II (Rise/Turn): Snow collapses hut; he takes refuge at mountain temple; overhears Lu Qian's arson plot.
Act III (Climax): Hearing truth, rage consumes him; bursts from temple and slays Lu Qian and men in blizzard.
Act IV (Resolution): Walks away with spear, vanishes into endless snow—the outlaw path begins.
```

## Step 4: Script Body Writing (Core)

### 4.1 Standard Shot-by-Shot Script Format

**Each episode must follow this format:**

```
Episode X
X-X [DAY/NIGHT] [INT/EXT] [Scene name]
Props: [prop1], [prop2]
Characters: [character1], [character2]

△ 【Establishing shot】[Opening atmosphere shot—detailed frame, camera, light]
△ [Shot 2—specific camera + frame + action]
△ [Shot 3—plot continues]
Character (OS): [interior monologue / narration]
△ [Reaction shot—face, body language]
Character: [dialogue]
△ [Action sequence—chain with →]
【Subtitle: xxx】[when on-screen text needed]
△ 【Flashback】[flashback scene]
[Flashback shots]
【Flashback end】
△ [Return to present—emotional reaction]
Character (VO): [off-screen voice, character not in frame]
△ [Climax action shot]
△ 【Establishing shot】[closing atmosphere shot]
【Subtitle: xxx】[closing subtitle]
```

### 4.2 Shot Description Quick Reference

#### Camera Movement Terms

| Movement | English | Use | Example |
|------|------|------|------|
| Push In | push in | Emphasize emotion, focus subject | Push to Lin Chong face close-up |
| Pull Out | pull out | Reveal environment, loneliness | Pull back over endless snowfield |
| Pan | pan | Reveal full scene | Pan left to right across ruins |
| Truck/Track | truck | Lateral follow | Truck with running figure |
| Follow Shot | follow shot | Follow subject | Follow Lin Chong walking |
| Orbit/Rotate | orbit shot | 360° showcase | Orbit around Lin Chong |
| Crane | crane shot | Dive or rise | Crane down to ground |
| Hitchcock Zoom | Hitchcock zoom | Background compression, pressure | |
| One Shot | one-shot | No cut | |
| Handheld | handheld | Documentary tension | |

#### Shot Size Terms

| Shot size | Frame range | Use |
|------|----------|------|
| Extreme wide | Full environment, tiny figures | Establish scene, mood |
| Wide | Full body | Action, spatial relation |
| Medium | Waist up | Common dialogue |
| Close-up | Chest up | Emotional exchange |
| Extreme close-up | Face or detail | Emotion, key info |
| Macro close-up | Eyes / hands | Peak emotion, clue |

### 4.3 Dialogue Notation

| Notation | Meaning | Example |
|------|------|------|
| `Character (OS)` | Off-screen / interior monologue | Lin Chong (OS): This world… |
| `Character (VO)` | Voiceover, not on screen | Lu Qian (VO): Once this fire is lit… |
| `Character (emotion)` | Emotional dialogue | Lin Chong (roaring): Traitors! Die! |
| `Character` | Normal dialogue | Lin Chong: I'll shelter at the temple tonight. |

### 4.4 Special Element Tags

| Element | Use | Example |
|------|------|------|
| `【Establishing shot】` | Atmosphere, no characters | △ 【Establishing shot】Ink-wash Cangzhou outskirts |
| `【Flashback】` | Memory | △ 【Flashback】Marshal Gao's son harassing wife… |
| `【Flashback end】` | Return to present | 【Flashback end】 |
| `【Subtitle】` | On-screen text | 【Subtitle: Cangzhou Hay Depot — Guard Lin Chong】 |

### 4.5 Action Sequence Description

**Use arrow → for chained action:**

```
❌ Wrong: Lin Chong killed three people

✅ Correct:
△ Lin Chong charges like a wrathful god; spear flashes cold.
Lin Chong (roaring): Traitors! Die!
△ Lu Qian staggers back → Lin Chong spears jail warden, blood on white snow → Fu An flees, legs cut, wails in snow → Lu Qian kneels begging, face twisted.
```

### 4.6 Sensory Detail

**Strong scripts engage multiple senses:**

| Sense | Example vocabulary |
|------|-------------|
| **Visual** | Color (gray-white, dark red, bright red), light (side light, silhouette, fire glow), composition (diagonal, centered) |
| **Auditory** | howling wind, crunching snow, crackling fire, heavy, urgent, wailing |
| **Tactile** | biting cold, frozen snow, burning rage |
| **Emotion** | lonely, uneasy, oppressed, eruption, resolve |

## Step 5: 15-Second Episode Rhythm Control

### 5.1 Shots Per Episode

| Episode type | Shot count | Pace | Best for |
|----------|--------|------|----------|
| Emotion / dialogue | 3–4 shots | 4–5s/shot | Inner life, talk, buildup |
| Action / fight | 5–7 shots | 2–3s/shot | Combat, chase, climax |
| Montage / sequence | 6–8 shots | 2s/shot | Flashback, fast cuts |

### 5.2 Emotional Arc

Every 15-second episode needs emotional movement:

```
0-3s: Establish mood / atmosphere
   ↓
3-9s: Rising tension / conflict develops
   ↓
9-12s: Peak / key reveal
   ↓
12-15s: Release / cliffhanger end
```

### 5.3 Episode Handoff

**Ending frame description must be detailed:**

```
✅ Correct:
**Scene:** In a courtyard, a child runs toward a large water jar in the corner.
**Frame:**
- Subject: 5–6 year old boy (light brown short jacket), running
- Background: Large ceramic jar (dark brown, calm water surface)
- Light: Spring afternoon sun from upper right
- Composition: Boy left, jar right, diagonal
- Mood: Ordinary moment hiding danger

**Purpose:** Opens E02, bridging to the instant before the accident.

❌ Wrong:
Lin Chong walks toward the mountain temple.
```

## Step 6: Visual & Audio Design

### 6.1 Color Planning

| Phase | Dominant palette | Emotional goal | Example (Lin Chong) |
|------|--------|----------|-------------|
| Opening | Gray-white, ink black | Lonely oppression | E1: cold gray-black isolation |
| Development | Warm yellow, gray-blue | Hidden danger | E3: fire glow vs cold exterior |
| Climax | Dark red, bright red | Rage / blood | E5: red blood on white snow |
| Ending | Vast white | Lingering resonance | Sky and earth, lone footprints |

### 6.2 Visual Motifs

Pick 1–3 visual elements that recur:

| Motif | Appears in | Symbolism |
|----------|----------|----------|
| Wind and snow | Throughout | Ruthless fate, environmental pressure |
| Long spear | E1 / E5 | Guard duty weapon → instrument of revenge |
| Wine gourd | E1 / E3 / E5 | Escape vs facing reality |

### 6.3 Sound Design

**Each episode uses three layers:**

```
Sound design:
- Music: [score style] + [instruments] + [rhythm]
- SFX: [ambience] + [action SFX] + [special SFX]
- Dialogue: [lines] (if any)
```

**Example (Lin Chong E4):**
```
Sound design:
- Music: Low oppressive drums, gradually accelerating like heartbeat
- SFX: Wind outside door, oil splash, fire, teeth grinding
- Dialogue: Lu Qian conspiracy (voiceover)
```

## Full Format Comparison

### ❌ Simple outline (old version — not acceptable)

```
Episode 1: Hay depot in snow
Key beats:
- Blizzard at Cangzhou hay depot
- Lin Chong guards alone, desolate
- Wind and snow, bleak environment
```

### ✅ Standard shot-by-shot format (required)

```
Episode 1
1-1 DAY EXT. Hay Depot
Props: long spear, wine gourd
Characters: Lin Chong

△ 【Establishing shot】Ink-wash Cangzhou outskirts, snow like salt, world black and white.
△ Extreme wide overhead, broken huts on snowfield, hay stacked like hills.
△ Push in; Lin Chong in felt hat and worn straw cloak, leaning on hay pile.
△ Lifts head, eyes exhausted; breath vapor vanishes.
Lin Chong (OS): Serve my sentence, survive this winter, return to the capital…
△ Gust hits; snow slaps face; he pulls cloak tighter.
△ Shakes wine gourd—empty.
△ Wind howls like a beast; depot flags snap.
【Subtitle: Cangzhou Hay Depot — Guard Lin Chong】
△ Takes spear, walks into snow, lonely silhouette.
```

## Quick Checklist

Before finishing the script, confirm:

### Format
- [ ] Every shot starts with `△ `
- [ ] Camera terms are specific (push / pull / pan / truck / follow / orbit / crane)
- [ ] Dialogue tagged correctly (OS / VO / emotion)
- [ ] Sensory detail included (visual + auditory + tactile)

### Content
- [ ] Each episode has clear emotional arc
- [ ] Episodes connect naturally (end frame → opening frame)
- [ ] Character consistent and develops
- [ ] Core conflict resolved
- [ ] Theme expressed

### Technical
- [ ] Each episode fits ~15 seconds (3–7 shots)
- [ ] End frame detailed (scene + frame + light + composition + mood)
- [ ] Color plan defined
- [ ] Sound design complete (music + SFX + dialogue)
