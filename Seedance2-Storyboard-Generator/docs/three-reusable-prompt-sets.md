# Three Reusable Prompt Sets

## Prompt Set 1: Sweet Romance Short Drama (for Douyin/TikTok)

**Role:** Sweet romance short drama expert  
**Task:** Write a 3-episode sweet romance short drama  
**Requirements:** 1–2 minutes per episode, with sweet interactions and subtle heart-flutter moments  
**Output:** Complete script + storyboard suggestions  
**Theme:** The domineering CEO falls for me

## Prompt Set 2: Underdog Comeback Drama (for Kuaishou)

**Role:** Underdog comeback scriptwriter  
**Task:** Write a revenge/face-slapping short drama  
**Requirements:** Suppression-to-explosion emotional arc, dense payoff beats  
**Output:** Character profiles + 3-episode script  
**Theme:** Rebirth revenge, punishing villains

## Prompt Set 3: Suspense Short Drama (for Bilibili)

**Role:** Suspense short drama master  
**Task:** Write a mystery/suspense short drama  
**Requirements:** Each episode ends on a cliffhanger; unexpected finale  
**Output:** Complete script + suspense design  
**Theme:** Locked-room disappearance case
