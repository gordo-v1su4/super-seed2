# Sima Guang’s Tank Smashing Project - User Guide

## Project Overview

This project is an AI video production script for the story of "Sima Guang Smashes a Vat". It includes a complete script, material list and storyboard script, and is suitable for the Seedance 2.0 video generation platform.

**Story duration**: 90 seconds (6 episodes × 15 seconds)
**Style Positioning**: Traditional Chinese ink animation style + modern animation color
**Cultural background**: During the Northern Song Dynasty, there is an allusion about Sima Guang’s wit in saving people when he was young.

## Production process

### Step one: Material generation

Use Nana Banana Pro or other AI image generation tools to generate the required materials according to the prompt words in the material list.

**Prioritization**:
1. **Must generate** (core material):
   - C01-C04: Sima Guang’s four states
   - C05: Basic appearance of partner children
   - S01-S09: all scene materials
   - P01-P03: key props

2. **Optional generation** (as needed):
   - C06-C09: Expression status of other children
   - P04-P05: Clothing details display

**Generation Notes**:
- Ensure consistent style: use the same style prefix for all materials
- Keep the character consistent: Sima Guang’s image should be consistent in each material
- Coherence of props: water tanks and stones have the same appearance in different scenes

### Step 2: Video generation

Videos are generated by episode using the Seedance 2.0 platform.

**E01Generation process**:
1. Upload S01, C01, C05, S03 to the material location
2. Copy Seedance Prompt in E01 storyboard
3. Set the duration to 15 seconds
4. Generate video

**E02-E06 generation process**:
1. Upload @video_file_1 (previous episode video) and corresponding picture materials
2. Copy the Seedance Prompt of the corresponding episode number (note: E02-E06 all start with "Extend @video_file_1 by 15s")
3. Set the duration to 15 seconds
4. Generate video

**IMPORTANT NOTE**:
- E02-E06 must use the "Video Extension" feature to maintain continuity
-Upload up to 9 pictures per episode
- Make sure the material number corresponds to the upload location in the storyboard

### Step 3: Video editing (optional)

Seedance 2.0 has limited video editing functions, and most modifications require regeneration. To edit:
1. Download the generated 6 episodes of video
2. Use external editing software (such as Cutting, Premiere) for splicing
3. Add opening, ending, subtitles, etc.

## FAQ

**Q: What should I do if the generated video styles are not uniform? **
A: Make sure all materials use the same style prefix and regenerate inconsistent materials.

**Q: The transition between the two episodes is not smooth? **
A: Check whether the @video_file_1 extension function is used correctly. The last frame description should be connected with the opening of the next episode.

**Q: Character appearance is inconsistent in different episodes? **
A: This is a common problem generated by AI, which can be improved by adding character descriptions to the prompts in each episode.

**Q: The generation of a certain episode failed? **
A: Check whether there are sensitive words (such as "charlatan") and try to modify the expression. Whether the number of materials exceeds the limit (maximum 9 pictures).
