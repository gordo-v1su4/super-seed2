# Chunk 03 — finale beats

## Abstract music video mode (default)

This is a **music video**, not a disaster doc. Beats can be **symbolic, dreamlike, and emotionally true** without literal physics. The viewer should **feel**: edge → ground gives → last hold → flash → release — not audit the geology.

| Feel | Abstract read |
|------|----------------|
| Rim walk | Lost in storm, drawn toward light below |
| Edge | Threshold between woods and abyss |
| Give-way (~10s) | Earth **dreams away** under her — mud/rock drift like ink in water |
| Grab / hang (~11s) | **Suspended** moment — tree as last tether; city opens below like memory |
| Final phrase (~12–14s) | Lightning = **flash cut** in the feeling; **release** into fall |
| Chunk 04 | The drop and river — overlap OK |

**One fluid camera move** through the finale — not four literal cuts. Slow motion and surreal timing are features, not bugs.

---

## Literal mode (optional reference)

If a gen needs more grounding, see compressed timing below. Use when abstract reads too vague — not required for every regen.

### Film touchstones (optional)

*The Impossible* (undercut), *Cliffhanger* (single hold), *127 Hours* (shear not explosion) — borrow **feeling**, not shot-for-shot realism.

### Compressed timing (sync to Video 1)

| Time | Audio | Literal action |
|------|-------|----------------|
| **0–10s** | Phrases 1–2 | Rim → notch → at edge, hand on tree |
| **~10–12s** | Third phrase | Lip slides; tree tips; she grabs |
| **~11.5–12.5s** | Overlap | Brief hang — one beat |
| **~12–14s** | Final phrase | Lightning flash → sheer → fall begins |
| **14s+** | Chunk 04 | Long drop, impact |

---

## Prompt wording (safe)

- "Abstract give-way — rim earth slides like a slow dream"
- "Suspended beat — tree only tether over drowned city"
- "One surreal passage — flash and release into fall"
- **Avoid:** lyric words, trunk explosion, megalog bridge, documentary avalanche

## Gens gone wrong

| Wrong | Push |

| Trunk explodes | Flash + release |
